## Reference Answer

`requirements.txt` is a file listing project dependencies (often with versions).
It prevents dependency/version conflicts across projects.
It helps others recreate the same environment, including classmates and CI.
It improves teamwork and reduces "works on my machine" problems.
It also helps maintenance because version changes are explicit.

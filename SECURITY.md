# Security Policy

## Reporting a Vulnerability

If you discover a security issue, please do not open a public issue with exploit details.

Instead, report it privately to the maintainers through GitHub Security Advisories (preferred) or by contacting the repository owner directly.

Please include:

- A clear description of the issue
- Steps to reproduce
- Affected versions/commits
- Potential impact

We will acknowledge receipt and work on a fix as quickly as possible.

## Supported Versions

This project is currently best-effort maintained on the latest `main` branch state.

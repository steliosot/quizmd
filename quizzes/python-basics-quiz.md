# Python Basics Quiz

## Data Type
Which type is returned by `len("quiz")` in Python?

- str
- int
- float
- bool

Answer: 2
Type: single
Time: 20
Explanation: `len` returns an integer.

## Mutable Collections
Which Python collections are mutable?

- list
- tuple
- dict
- str

Answer: 1,3
Type: multiple
Time: 30
Explanation: Lists and dictionaries are mutable.

## Loop Keyword
Which keyword exits a loop immediately?

- continue
- break
- pass
- return

Answer: 2
Type: single
Time: 20
Explanation: `break` exits the nearest enclosing loop.

## Boolean Expressions
Which expressions evaluate to `True`?

- 3 > 2

Time: 30
